Suppression de compte

Pour supprimer votre compte, suivez les étapes suivantes :

  1. Accédez à la page de gestion de votre compte en naviguant vers l'onglet "Compte" dans l'application.
  2. Sur la page de gestion de votre compte, faites défiler jusqu'en bas pour trouver l'option "Supprimer votre compte".
  3. Cliquez sur le bouton "Supprimer votre compte" et suivez les instructions à l'écran pour confirmer la suppression.
  4. Une fois la suppression confirmée, vous recevrez une notification de confirmation.
